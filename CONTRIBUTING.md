# Contributing

## Development

Use Node 20, 22, or 24 for local development. The repository has
`engine-strict=true`, and the test tooling follows the same even-numbered Node
support window.

```bash
pnpm install
pnpm typecheck
pnpm test
pnpm build
```

## Release Checklist

Before publishing:

1. Run `pnpm audit --audit-level moderate`.
2. Run `pnpm typecheck`.
3. Run `pnpm test`.
4. Run `pnpm build`.
5. Run `env npm_config_cache=/tmp/citekit-npm-cache npm pack --dry-run`.
6. Inspect the package file list and confirm it includes `dist/`, `docs/`,
   `styles/`, `venues/`, `README.md`, `LICENSE`, and `package.json`.

`dist/` is intentionally ignored by git. It is generated by `pnpm build` and
included in the npm package through the `files` allowlist in `package.json`.
Always build immediately before packing or publishing.
